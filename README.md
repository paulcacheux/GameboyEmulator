Gameboy Emulator
================

## Usage
```
$ ./gameboy_emulator <BOOTSTRAP_ROM> <GAME_ROM>
$ ./gameboy_emulator <GAME_ROM>
```


## Joypad Mapping

| Gameboy | Emulator       |
|---------|----------------|
| Arrows  | Arrows or ZQSD |
| A       | O              |
| B       | P              |
| Start   | Enter/Return   |
| Select  | Left Control   |
